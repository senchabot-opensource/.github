## Hi there 👋

Welcome to **Senchabot Open Source** — a community-driven project building open source bots and tools for [Twitch](https://twitch.tv) and [Discord](https://discord.com).

---

### 🤖 What is Senchabot?

Senchabot is a feature-rich, open source bot platform that lets streamers and community managers automate their Twitch and Discord servers. It includes a web dashboard so you can manage everything from your browser.

---

### 📦 Our Projects

| Repository | Description |
|---|---|
| [monorepo](https://github.com/senchabot-opensource/monorepo) | Core monorepo — Twitch bot, Discord bot, and web dashboard (Go + TypeScript) |
| [docs](https://github.com/senchabot-opensource/docs) | Official documentation site (VitePress + Vue.js) |
| [discussions](https://github.com/senchabot-opensource/discussions) | Community discussions, ideas, and feedback |

---

### 🛠️ Tech Stack

- **Backend / Bots:** Go
- **Web Dashboard:** Next.js, React, Tailwind CSS, shadcn/ui
- **Docs:** VitePress, Vue.js
